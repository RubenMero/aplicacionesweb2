Descripcion del taller

Este taller practico consiste en el desarrollo de una calculadora cientifica en consola (CLI) utilizando el lenguaje de programacion Go. 
Permite realizar operaciones basicas (suma, resta, multiplicacion y division) y operaciones mas avanzadas como potencia y factorial, incluyendo
validacion de errores y un historial de operaciones.

El objetivo es aplicar conceptos fundamentales como variables, tipos de datos, estructura de control, bucles y entrada/salida por consola.
